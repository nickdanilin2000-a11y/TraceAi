# TraceAI — Продуктовая документация

## Что это такое
TraceAI — корпоративная платформа для работы с AI.
Все AI-запросы сотрудников логируются автоматически:
кто спросил, что ответила модель, кто проверил, кто утвердил.

## Для кого
Компании 50–2000 сотрудников, которые используют AI
в ежедневной работе и хотят контролировать это использование.

## Экраны платформы

### 1. Дашборд
Главный экран для руководителя.
- Количество AI-запросов за день
- Активность команды
- Стоимость токенов
- Лента последних событий

### 2. Редактор документов
Создание документов с AI-ассистентом.
- Синим выделен AI-текст
- Оранжевым — правки человека
- Статусы: Черновик → На проверке → Утверждён → Отправлен
- История каждого изменения с именем и временем

### 3. AI-Чат
Корпоративный чат вместо личных ChatGPT аккаунтов.
- Все диалоги логируются
- Показывает количество потраченных токенов
- История разговоров доступна руководителю

### 4. Аналитика
- График использования по дням
- Активность сотрудников
- Стоимость по моделям
- Популярные типы задач

### 5. Audit Trail
Полная история всех AI-событий.
- Фильтры по типу, пользователю, дате
- Экспорт в PDF
- Для внутреннего аудита и проверок
